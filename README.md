Post mejorado (save)
https://misautos2.herokuapp.com/api/{nombre de entidad}

Put (update)
https://misautos2.herokuapp.com/api/{nombre de entidad}

Delete (delete)
https://misautos2.herokuapp.com/api/{nombre de entidad}
